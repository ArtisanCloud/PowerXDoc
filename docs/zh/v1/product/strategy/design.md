# 战略设计

战略设计的理论非常多，都是从不同角度树立战略的。我们应该了解不同理论产生的背景、适用的场景和解决的问题。

大致分为四种：

* [通用分析模型](design/general.md)
* [从竞争者角度分析](design/competitor.md)
* [从差异化角度分析](design/differentiation.md)
* [以及破坏性的创新](design/destructive.md)
