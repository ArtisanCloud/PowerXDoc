# LBEE 立特缤生鲜甄选

LBEE CluB生鲜甄选 是上海立特缤网络科技有限公司的一个高端生鲜产品品牌，致力于向追求生活品质的中高端家庭提供精品生鲜食材。秉承只想给您更好的品牌理念， 严选原产地和生鲜食材，遵循严格的农药残留及食品安全检测标准，精选近千种生鲜，并通过专业、高效的冷链运输， 将更优质的生鲜食材直达消费者手中。
